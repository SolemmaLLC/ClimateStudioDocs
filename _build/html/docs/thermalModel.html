Thermal Model¶

Climate Studio supports multi zone thermal simulations using the US Department of Energy’s EnergyPlus whole building simulation program. The workflow has two simulation subpanels plus a detailed Manage Library menu and EnergyPlus Simulation Settings.

  • The Location Subpanel

  • The Add Objects Subpanel

  • The Manage Library Menu

  • The EnergyPlus Simulation Settings

Once all required input subpanels have been populated, a simulation is invoked by pressing the start button. A DOS window should appear that is running the EnergyPlus simulation.
